HƯỚNG DẪN CHẠY TRÊN XAMPP
1. Sao chép (hoặc di chuyển) thư mục project này vào XAMPP htdocs (ví dụ: C:\xampp\htdocs\domic)
2. Khởi động Apache trong XAMPP Control Panel.
3. Truy cập http://localhost/domic/index.php
4. Demo: tạo user bằng trang /auth/register.php
5. Dữ liệu tin đăng lưu tạm ở /data/listings.json
